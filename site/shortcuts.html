Morning Masterpieces — Shortcuts

A new masterpiece on your lock screen every morning, and a docent you can summon with your voice. Two ways to build each shortcut: ask Siri — copy a blurb below and read or paste it to Siri in the Shortcuts app — or follow the manual steps.

← Back to the gallery

One-time setup, before anything else: the daily wallpaper needs a dedicated wallpaper slot to overwrite. Long-press your lock screen → + → create a plain Photo wallpaper from any image (name it — e.g. “Masterpiece”). It must be a Photo wallpaper, not Photo Shuffle: Shortcuts can’t modify a Shuffle wallpaper. The blurbs below call it “Masterpiece” — swap in whatever yours is named.

Shortcut 1 · iPadDaily wallpapers, lock & home

Swaps both screens each morning: the matted, captioned composition on the lock screen, the painting edge-to-edge behind your icons.

Create a new shortcut named "Morning Masterpiece" with exactly four actions. Action 1: "Get Contents of URL" fetching https://thirdbrainrepo.github.io/morning-masterpieces/today/wallpaper-ipad.jpg as a GET request. Action 2: "Set Wallpaper" using the output of action 1 as the image, targeting the Lock Screen of my wallpaper named "Masterpiece", with the "Show Preview" option turned off. Action 3: "Get Contents of URL" fetching https://thirdbrainrepo.github.io/morning-masterpieces/today/home-ipad.jpg as a GET request. Action 4: "Set Wallpaper" using the output of action 3 as the image, targeting the Home Screen of that same wallpaper, with "Show Preview" turned off. Then create a personal automation that runs this shortcut daily at 6:00 AM, set to "Run Immediately" with Notify turned off.

Build it by hand instead
  1. Shortcuts → + → name it Morning Masterpiece.
  2. Get Contents of URL → https://thirdbrainrepo.github.io/morning-masterpieces/today/wallpaper-ipad.jpg
  3. Set Wallpaper → image: Contents of URL → pick your art wallpaper → Lock Screen → expand the action’s options and turn Show Preview OFF.
  4. Get Contents of URL → …/today/home-ipad.jpg
  5. Set Wallpaper → image: output of step 4 → same wallpaper → Home Screen → Show Preview OFF.
  6. Automation tab → + → Time of Day → 6:00 AM, Daily → Run Immediately → run Morning Masterpiece.

Shortcut 3 · any device“Hey Siri, Art Docent”

Opens the artwork fullscreen and reads today’s lesson aloud in the Siri voice — including the advanced expressive voice on devices that have it.

Create a new shortcut named "Art Docent". Action 1: "Get Contents of URL" fetching https://thirdbrainrepo.github.io/morning-masterpieces/today.json. Action 2: "Get Dictionary Value" for key path "item.title" from the contents. Action 3: "Get Dictionary Value" for key path "item.artist". Action 4: "Get Dictionary Value" for key path "item.lesson". Action 5: "Get Dictionary Value" for key path "item.lookFor". Action 6: "Open URL" opening https://thirdbrainrepo.github.io/morning-masterpieces/?view=full. Action 7: "Wait" 2 seconds. Action 8: a "Text" action composing: the title, followed by ", by ", the artist, then a period, then the lesson, then " Look closer: " and the lookFor value. Action 9: "Speak Text" speaking that composed text aloud, with "Wait Until Finished" turned on.

Build it by hand instead
  1. Get Contents of URL → https://thirdbrainrepo.github.io/morning-masterpieces/today.json
  2. Get Dictionary Value ×4 → key paths item.title, item.artist, item.lesson, item.lookFor.
  3. Open URL → https://thirdbrainrepo.github.io/morning-masterpieces/?view=full (the gallery opens straight into the fullscreen artwork).
  4. Wait → 2 seconds.
  5. Text → [title], by [artist]. [lesson] Look closer: [lookFor]
  6. Speak Text → the Text variable → turn Wait Until Finished ON. Pick the voice and rate in this action’s options.

After Siri builds a shortcut, check the two things it most often gets wrong:

  • Every Set Wallpaper action must have Show Preview OFF — otherwise iOS asks for confirmation every morning and the automation stalls.
  • The automation must be Run Immediately, not “Ask Before Running.”

Then run the shortcut manually once — iOS asks one-time permission to reach this site and to set wallpapers. After that it’s fully hands-off.
